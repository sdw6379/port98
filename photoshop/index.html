Photoshop Cheetsheet

  • 크기 변경하기:ctrl + T
  • 새 파일 만들기 : ctrl + N
  • 레이어 패널 :F7
  • 작업 취소하기 : Ctrl + Z
  • 이미지 불러오기 : Ctrl +O
  • 화면 100% 비율 조정 : Ctrl +1
  • 화면 확대 : Ctrl + +
  • 화면 축소 : Ctrl + -
  • 환경 설정 : crel + K
  • 컴퓨터 사양보기 : F8 > Panel Options> Efficiency
  • 화면 분활: Widow >Arrange
  • 전경색 바꾸기 : Alt + Delete
